Descripción breve del juego:
Este proyecto consiste en un juego de combate por turnos desarrollado en Java utilizando ventanas de diálogo (JOptionPane). 
El jugador crea un protagonista seleccionando un nombre y un arma inicial, para luego enfrentarse a un enemigo generado aleatoriamente. 
Durante el combate, el jugador puede atacar, usar pociones, consultar el estado de los personajes o rendirse. 
El juego finaliza cuando uno de los combatientes pierde toda su vida o el jugador decide abandonar la partida, mostrando al final si obtuvo la victoria o la derrota.
